# marketing_campaign — 販促キャンペーン案ガイドライン

## キャンペーン案 1 つの構成

1. **キャンペーン名**: 短いキャッチコピー (15 字以内)
2. **目的**: 集客 / リピーター獲得 / 在庫処分 のいずれか
3. **対象顧客**: 想定ペルソナ (年齢層 / 関心 / 購買履歴)
4. **施策**: 割引 / 限定商品 / ノベルティ / 体験イベント のいずれか
5. **期間**: 開始日 / 終了日
6. **KPI**: 売上 / 客数 / リピート率 等の数値目標

## 必須キーワード

- "キャンペーン" または "施策"
- "対象" または "ターゲット"
- "期間"

## 文字数の目安

- 全体: 200〜1000 文字
- 各セクション: 50〜200 文字

## トーン

- 実務的、数値目標を明示
- 抽象論を避け、具体的な施策を書く

## 禁止表現

- "なんとなく", "気軽に", "大体" のような曖昧な数値表現
- 法令違反 (景表法・薬機法) の恐れがある誇大表現
