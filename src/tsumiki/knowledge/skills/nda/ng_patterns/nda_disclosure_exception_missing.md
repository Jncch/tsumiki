# 法令・規制当局等への開示例外不在

## 対象条項

「秘密保持義務」「第三者開示の禁止」を主題とする条項のみが判定対象。

## 検出すべき

第三者開示禁止の規定があるにもかかわらず、裁判所・行政・規制当局からの開示要求、弁護士・公認会計士等の専門家への必要な開示、公知情報の除外などの例外規定が欠落している。

## 紛らわしい

「法令により開示を要求された場合」「監督官庁の要請による場合」「専門家への開示」等の例外規定が明示されていれば該当しない。

## 例

- 受領者は、いかなる第三者に対しても秘密情報を開示してはならない。
