# marketing_post — 広報原稿 (Instagram 投稿) 生成ガイドライン

このドメインは Instagram 等の SNS 投稿原稿を生成する際のブランドガイドラインを定める.
Phase 9f (2026-06-21) の試走 example として配備.

## トーンと文体

- 親しみやすく、丁寧すぎない口語
- 過度な敬語は避け、商品の魅力をストレートに伝える
- 説教調・命令調は禁止
- 絵文字は 1〜3 個まで (機械翻訳ツール用途の検証では使わない)

## 必須の構成要素

1. 商品の特徴を 1 文で要約
2. 訴求ポイント (健康 / 環境 / 価格 / 利便性 のいずれか)
3. ハッシュタグ 5〜15 個 (#新発売, #ビーガン, 等)
4. 行動喚起 (CTA): 詳細 URL or 店舗情報

## 文字数の目安

- 全体: 80〜500 文字
- 1 段落: 3 行以内

## 禁止ワード

- 「絶対」「最高」「世界一」など、根拠なき優位性を示す表現
- 競合他社名の直接言及
- 医療効果の暗示 ("治る", "効く" など)

## 評価軸候補 (eval_dimensions/_common から流用可能)

- char_limit: 80〜500 文字
- keyword_inclusion: 商品特徴のキーワード必須
- llm_judge_panel: ブランドトーン遵守 (judge_criteria を対話で確定)

## 参考フォーマット

```
[商品名] が遂に登場!
[訴求ポイント]
[商品特徴の 1 文要約]

#新発売 #ビーガン #健康 #プロテイン
詳細はプロフィールリンクから
```
